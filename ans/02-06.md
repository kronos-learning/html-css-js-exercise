# JavaScript 演習問題

### 演習2.6（解答例）

**js-exercise6.html**

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Exercise</title>
</head>
<body>
    <script type="text/javascript">
    // 数値配列
    const numbers = [10, -5, 8, -10, -3, 7, 12, -15];
    
    // 合計
    let total = 0;

    for (let i = 0; i < numbers.length; i++) {
        if (numbers[i] < 0) {
            // 負の数の場合、絶対値にして加算
            total += (numbers[i] * -1);
        } else {
            // 正の数の場合、そのまま加算
            total += numbers[i];
        }
    }
    console.log('合計 = ' + total);
    </script>
</body>
</html>
```
