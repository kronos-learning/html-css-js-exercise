# JavaScript 演習問題

### 演習2.8（解答例）

**js-exercise8.html**

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Exercise</title>
</head>
<body>
    <script type="text/javascript">
    for (let i = 1; i <= 5; i++) {
        // 星（空で初期化）
        let star = '';
        for (let j = 1; j <= i; j++) {
            // 行番号分、星を結合する
            star += '★';
        }
        console.log(star);
    }
    </script>
</body>
</html>
```
