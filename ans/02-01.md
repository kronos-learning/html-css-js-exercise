# JavaScript 演習問題

### 演習2.1（解答例）

**js-exercise1.html**

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Exercise</title>
</head>
<body>
    <script type="text/javascript">
    let x = 10;
    let y = 20;
    let z = 30;

    let temp = y;
    y = x;
    x = z;
    z = temp;

    console.log('x = ' + x);
    console.log('y = ' + y);
    console.log('z = ' + z);
    </script>
</body>
</html>
```
