# JavaScript 演習問題

### 演習2.5（解答例）

**js-exercise5.html**

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Exercise</title>
</head>
<body>
    <script type="text/javascript">
    // 数値配列
    const numbers = [32, 45, 11, 24, 63];
    
    // 最大値
    let max = 0;

    for (let i = 0; i < numbers.length; i++) {
        if (numbers[i] > max) {
            // 最大値より大きい場合、配列の値を代入する
            max = numbers[i];
        }
    }
    console.log('最大値 = ' + max);
    </script>
</body>
</html>
```
